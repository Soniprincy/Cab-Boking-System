# Cab-Boking-System

## Live app -> https://cab-booking-app.streamlit.app/ 

This is a Cab booking app, 
Which requires details like:
### 1. name
### 2. email
### 3. phone no.
### 4. pickup address
### 5. drop address
and it returns the invoice in table and we are able to download that invoices in pdf form from that app.
